# filldays

Clojure snipet that provided with a list of numbers returns the numbers and the gaps in-between filled

## Usage

    $ lein uberjar
    $ java -jar filldays-0.1.0-standalone.jar 1 3 5

    or

    $ lein run 1 3 5
